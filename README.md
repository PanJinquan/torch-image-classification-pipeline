# torch-image-classification-pipeline

## 1.目录结构

```
├── configs         # 配置文件
├── models          # 模型实现等相关文件
├── libs            # 相关模块依赖包
├── data            # 数据文件夹,包含测试数据和部分训练数据
├── docker          # docker文件
├── evaluation      # 测评工具
├── utils           # 项目常用的工具函数
├── work_space      # 工作空间
├── validation.py   # 测试文件
└── train.py        # 训练文件

```

## 2.train

```bash
bash train.py
```

## 3.validation

```bash
bash validation.py
```
